# Log Analyzer

## Overview
A Python-based log analyzer that detects failed login attempts and suspicious IP activity from authentication logs.

This project demonstrates practical cybersecurity skills used in SOC environments, including log analysis, brute-force detection, and basic incident investigation.

---

## Features
- Detects failed login attempts
- Extracts suspicious IP addresses
- Counts failed login attempts per IP
- Flags possible brute-force attacks
- Simple command-line interface

---

## Tech Stack
- Python
- Regex
- Collections Counter

- log_analyzer.py → main script
